<h2>🛠️ Installation Steps:</h2>

<p>1. 🔧 Clone the Repository</p>

<pre>
git clone https://github.com/ghssx19/RL-Project-Gym.git
</pre>

<p>2. Directory</p>

<pre>
cd multi_car_racing
</pre>

<p>3. Conda</p>

<pre>
conda env create -f conda_explicit.txt
</pre>

<p>4. Pip</p>

<pre>
pip install -r pip_frozen.txt
</pre>
